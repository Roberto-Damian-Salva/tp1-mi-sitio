# Proyecto: Prácticas Profesionales II - Unidad 2

## Descripción
Este proyecto consiste en una mini-app desarrollada con JavaScript vainilla que incluye una lista de tareas dinámica (To-Do List) y un buscador de usuarios consumiendo una API externa.

## Páginas del Proyecto
1. **Todo List (`todo.html`):** Permite agregar tareas, marcarlas como completadas con un tachado y eliminarlas. Incluye un contador dinámico de tareas pendientes.
2. **API Demo (`api-demo.html`):** Muestra una lista de usuarios traídos desde JSONPlaceholder.
3. **Buscador Dinámico:** Filtra los usuarios en tiempo real a medida que el usuario escribe (mínimo 3 caracteres).

## Tecnologías Utilizadas
* HTML5
* CSS3 (Variables, Flexbox y Grid)
* JavaScript ES6+
* Fetch API para el consumo de datos asíncronos.
* JSONPlaceholder (API elegida)
